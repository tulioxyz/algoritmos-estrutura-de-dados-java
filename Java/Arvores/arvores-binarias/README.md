# Atividade Prática: Árvore Binária de Busca

## Estrutura do Projeto
 
- Para cada questão, você deve implementar a função correspondente no arquivo `ArvoreBinariaDeBusca.java`.
- O arquivo `Principal.java` de cada pacote de questão (ex: `src/questao01/Principal.java`) deve ser utilizado para testar a função implementada. 

---

## Questões
1. Implemente os métodos básicos para uma árvore binária de busca, incluindo inserção, remoção e busca de elementos.
2. Implemente um método para percorrer a árvore em ordem (in-order traversal) e retornar os elementos em uma lista ordenada.
3. Implemente um método para calcular a altura da árvore binária de busca.
4. Implemente um método para verificar se a árvore é balanceada.
5. Implemente um método para encontrar o menor e o maior elemento na árvore binária de busca.
6. Implemente um método para imprimir a árvore em níveis (em largura).
7. Escreva uma funçãao que conte o número de nós de uma árvore binária.
8. Escreva uma função que conte o número de folhas na árvore binária de busca.
9. Escreva uma função que exclui todos os nós de uma árvore binária de busca com valor par.
10. Escreva uma função não recursiva verifique a existência de ao menos um valor negativo na árvore binária de busca.
11. Escreva uma função não recursiva que verifique a existência de um determinado valor na árvore binária de busca.
12. Implemente um método que retorne a soma de todos os valores armazenados na árvore binária de busca.
13. Implemente um método que retorne a média dos valores armazenados na árvore binária de busca.
14. Implemente uma função que retorne o caminho do nó raiz até um nó com um valor específico.
