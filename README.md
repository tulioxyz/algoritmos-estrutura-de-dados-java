# Bem-vindo ao repositório de Algoritmos e Estruturas de Dados em Java!

Aqui estão reunidos exercícios e práticas voltadas ao estudo de estruturas de dados utilizando Java.

## 🎯 Objetivo
Este repositório tem como objetivo:

- Praticar estruturas de dados  
- Entender organização e manipulação de dados  
- Melhorar a lógica de programação  
- Servir como base para estudos mais avançados  

## 🚀 Tecnologias utilizadas
- Java
